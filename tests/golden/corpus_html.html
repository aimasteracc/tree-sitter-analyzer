About This Page

This document exercises every named node type in the tree‑sitter HTML grammar. Entities used here include: & (ampersand), < (less-than), > (greater-than), " (double-quote), ' (apostrophe), © (copyright), ® (registered), © (decimal entity), © (hex entity).

Attributes appear in multiple forms: href="..." (double-quoted), lang='en' (single-quoted), and boolean attributes such as disabled and checked.
