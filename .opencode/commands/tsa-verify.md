# /tsa-verify

tree-sitter-analyzerのSpecと実装・テストの整合性を検証するコマンド。

## 使用方法

```
/tsa-verify [target] [options]
```

## 引数

| 引数 | 説明 |
|------|------|
| `[target]` | 検証対象 (省略時は全Spec) |

## オプション

| オプション | 説明 |
|-----------|------|
| `--spec` | Spec自体の妥当性検証 |
| `--impl` | Specと実装の整合性 |
| `--tests` | Specとテストのカバレッジ |
| `--all` | 全検証実行 |
| `--report` | 詳細レポート出力 |

## 実行例

```
/tsa-verify                           # 全Spec検証
/tsa-verify mcp.tools                 # 特定モジュール
/tsa-verify --all --report            # 全検証+レポート
/tsa-verify --tests                   # テストカバレッジのみ
```

## 検証項目

### Spec妥当性 (--spec)
- [ ] 概要が記載されている
- [ ] インターフェースが定義されている
- [ ] 振る舞いが記載されている
- [ ] 受け入れ基準が定義されている
- [ ] テストケースが網羅されている

### 実装整合性 (--impl)
- [ ] 関数シグネチャが一致
- [ ] パラメータ型が一致
- [ ] 戻り値型が一致
- [ ] 例外が適切に送出

### テストカバレッジ (--tests)
- [ ] 正常系テストが存在
- [ ] 境界値テストが存在
- [ ] 異常系テストが存在
- [ ] Specのテストケースと対応

## 出力例

